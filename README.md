## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Swagger
to access the swagger you can visit this [url](http://localhost:3000/v1/event/docs)

## Run and create the migration
```bash
# Create migration
$ typeorm:create-migration

# Run the migration
$ typeorm:run-migrations

```
